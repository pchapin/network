CIS-3152 Homework #4 (IPv6)

Due: Friday, March 12, 2010

This homework covers IPv6. The text book discusses the basics of IPv6 in Appendix A. There are also many IPv6 related RFCs---too many to mention them all here. Here are a few highlights.

  • RFC-2373: The format of IPv6 addresses.
  • RFC-2460: The base specification for IPv6.
  • RFC-2461: Neighbor discovery protocol.
  • RFC-2462: Address autoconfiguration.
  • RFC-2463: ICMPv6.
  • RFC-2646: Using IPv6 on Ethernet.
  • RFC-3056: 6to4 Tunneling.
  • RFC-4193: Unique local addresses.

Feel free to look over some of these RFCs although you might find the discussion in the text to be shorter and more approachable.

  1. IPv6 defines the concept of an "anycast" address. What does this mean? Can you think of an interesting application of such addresses? What problems can you see with using them? HINT: Think about what anycast addresses imply for routers. See RFC-2373.

  2. What is Classless Interdomain Routing (CIDR) and how does it relate to IPv6?

  3. Every link that will carry IPv6 needs to support a certain minimum MTU (Maximum Transmission Unit) size. What is that size? Provide a reference. The minimum MTU for IPv6 is larger than that for IPv4. Will this prevent IPv6 from being sent over some links currently in use? Explain.

  4. Look up "Teredo Tunneling" online and write a short description (one or two paragraphs) that describes what it is and why it is useful. Give a reference for any source that you use.

CIS-3152 Homework #5 (Ice)

Due: Friday, April 26, 2013

This homework covers Ice. For information on Ice, consult your class notes and the ZeroC web site.

  1. Write a demonstration Ice client program (in Java) that uses the fileservices module described in FileServices.ice to print a listing of the root directory for a particular file system. The listing should contain the name of each file or directory in the root directory along with its type. Assume a FileSystem object is available on the localhost at port 9111.
